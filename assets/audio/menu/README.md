# Tater Tube menu sounds

These original, synthesized interface sounds are generated by
`tools/generate-menu-sounds.sh` and are distributed under the same license as
Tater Tube.

- `soft-touch`: warm, restrained VCR transport buttons
- `rental-night`: crisp rental-era front-panel beeps and shuttle mechanics
- `haunted-tape`: worn heads, pitch flutter, and cassette noise

Each pack contains `move.wav`, `page.wav`, `back.wav`, and `select.wav`.
